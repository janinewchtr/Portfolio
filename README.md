# Portfolio

This is my personal frontend developer portfolio. It presents my skills, selected projects, contact information, and legal pages in a responsive bilingual website.

The portfolio was built with HTML, CSS, JavaScript, and a small PHP contact form. It includes project showcases for Join, Sharkie, and Pokédex, each with live demos and GitHub links.

## Features

- Responsive layout for desktop, tablet, and mobile
- German and English language switch
- Project showcase section with live links
- Contact form with validation
- Legal notice and privacy policy pages
